Project — Leap Year & Loops

import java.util.Scanner;

public class LoopsAndLeap {
  public static void main(String[] args) {
    // Leap Year Checker
    int year = 2024;
    if (isLeap(year)) System.out.println(year + " is a leap year.");
    else System.out.println(year + " is not a leap year.");

    // Simple loop demo
    Scanner scanner = new Scanner(System.in);
    int a = scanner.nextInt();
    for (int i = 0; i <= a; i++) System.out.print("*");
  }

  public static boolean isLeap(int year) {
    if (year % 400 == 0) return true;
    if (year % 100 == 0) return false;
    return year % 4 == 0;
  }
}

Reflection: The original version had typos in the class declaration, used = instead of ==, and misplaced parentheses in the leap year condition. The loop also missed semicolons and had Scanner syntax errors.

Project — Palindrome Checker

import java.util.Scanner;

public class PalindromeCheck {
  public static void main(String[] args) {
    Scanner scanner = new Scanner(System.in);
    int q = scanner.nextInt();
    String str = "" + q;

    String rev = new StringBuilder(str).reverse().toString();
    if (str.equals(rev)) {
      System.out.println("It's a palindrome!");
    } else {
      System.out.println("Not a palindrome.");
    }
  }
}

Reflection: The first handwritten attempt used invalid array indexing and == for string comparison instead of equals(). The Scanner syntax was also wrong (System,in).

Project — Comparison & Binary Converter

import java.util.Scanner;

public class CompareAndBinary {
  public static void main(String[] args) {
    Scanner scanner = new Scanner(System.in);

    // Compare three numbers
    int a = scanner.nextInt();
    int b = scanner.nextInt();
    int c = scanner.nextInt();
    if (a != 0 || b != 0 || c != 0) {
      System.out.println("Values are not zero, continue.");
    }
    if (b != 0 && c != 0 && (a / b) == (b / c)) {
      System.out.println("All are equal.");
    }

    // Binary to Decimal Conversion (4-bit)
    System.out.print("Enter a 4-bit binary number: ");
    String s = scanner.next();
    int total = 0;
    if (s.length() == 4) {
      char b3 = s.charAt(0), b2 = s.charAt(1), b1 = s.charAt(2), b0 = s.charAt(3);
      if (b3 == '1') total += 8;
      if (b2 == '1') total += 4;
      if (b1 == '1') total += 2;
      if (b0 == '1') total += 1;
      System.out.println("Decimal value: " + total);
    } else {
      System.out.println("Please enter exactly 4 bits (e.g. 1010).");
    }
  }
}

Reflection: The handwritten version used incorrect logical operators, double quotes instead of single quotes for chars, and symbols like ± or missing semicolons that broke compilation.
